# Virtual Private Networks
> allows for the extension of a private of local network to hosts that might not be on that local network

example: for employees to acces company's network but not in the office

- VPN using payload section of the packet.
- Usually require two factor auth.
- encrypted tunnels


# Proxy Services
> a server that acts on behalf of a client in order to access another service

Act as an intermediary between a client an a server.

Benefits
- anonymity
- security
- content filtering
- increased performance

Other points
- Gateway routers can be called as a proxy.
- exist in almost every layer of network
- web proxy: can direct web traffic
- reverse proxy: represents many servers as one server
	- uses round robin concept
